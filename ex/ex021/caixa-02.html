Tags da Aula

<header>

Significa "Cabeçalho" e é feita pra servir de cabeçalho (obviamente). Da pra colocar barras de pesquisas, menus ou até apresentações.

Explicação do google: A tag <header> serve para representar o cabeçalho de uma página web no HTML. É um elemento que pode conter vários elementos de cabeçalho, como: Um ou mais elementos de título (<h1> - <h6>), Um logotipo ou ícone, Link de navegação, Um formulário de pesquisa.

<nav>

"Navegation" significa navegação e é pra ser uma seção de navegação pelo site.

Explicação do google: A tag <nav> do HTML5 serve para indicar uma seção de uma página com links de navegação, apontando para outras páginas ou para outras áreas da página

<main>

Significa "Principal", é o conteudo principal do site.

Explicação do google: A tag <main> do HTML serve para identificar o conteúdo principal de uma página web. Ela deve ser usada dentro do elemento <body> e deve ser única na página.

<section>

Significa "seção" e serve pra agrupar informação ou tags, numa vibe "contêiner" das ideia.

Explicação do google: A tag <section> do HTML5 serve para marcar seções de conteúdo em uma página, agrupando-as de forma lógica e separando a informação em áreas diferentes.

<article>

Significa "artigo" e eu n entendi muito bem as funcionalidades disso, vai ver sou mei burro

Explicação do google: A tag HTML <article≶ serve para marcar um conteúdo que é independente em uma página, aplicação, site ou documento. Esse conteúdo pode ser reutilizado ou distribuído de forma isolada, como por exemplo em sindicação. Alguns exemplos de conteúdos que podem ser marcados com a tag <article> são: Post em um fórum, Artigo de uma revista ou jornal, Matéria em um blog, Comentário de um usuário, Widget ou gadget interativo.

<aside>

Significa "a parte" "de lado", e pelo o que eu entendi é um conteudo que tem relação com o conteúdo principal.

<footer>

Significa "rodapé" e é o lugar pra colocar informações sobre quem fez o site ou da empresa.

Explicação do google: A tag <footer> em HTML serve para marcar a área inferior de uma página, ou seja, o rodapé. O rodapé é normalmente utilizado para incluir informações como:

  • Endereço
  • Contato
  • Link pra páginas importantes
