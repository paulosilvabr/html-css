Hierarquia de Formas de CSS

Se caso voce colocar uma forma de CSS por cima de outra, dependendo de qual forma voce colocar ela vai se sobrepor a outra, Por ordem de "mais forte" (de cima para baixo) eu fiz essa lista para demonstrar:

  1. Inline (No <body>)
  2. Interno (No <head>)
  3. Externo (Em outro arquivo separado: <link:css> no body)
