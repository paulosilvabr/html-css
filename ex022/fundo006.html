Conceito Símbolo Exemplo O que faz?
Pseudoclasse : a:hover Aplica estilo baseado no estado do elemento (ex: quando o mouse passa em cima).
Pseudoelemento :: p::first-letter Permite estilizar partes específicas de um elemento (ex: primeira letra de um parágrafo).
Seletor Universal * * { margin: 0; } Seleciona todos os elementos da página.
Charset @ @charset "UTF-8"; Define a codificação de caracteres do CSS para suportar acentos e símbolos.
:root (variáveis) :root :root { --cor: azul; } Declara variáveis CSS globais que podem ser reutilizadas no estilo da página.
