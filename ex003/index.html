Testando Carga e Imagens

Abaixo Voce vai Ver uma imagem que esta na mesma pasta do arquivo code.

Pode-se Colocar imagens que estao em sub-pastas (Consideirando que a pasta raiz é a pasta que o arquivo HTML está).

Também podemos carregar imagens direto da internet usando o URL dela (Se caso o servidor onde essa imagem esta hospedada cair, cairá no seu também).
